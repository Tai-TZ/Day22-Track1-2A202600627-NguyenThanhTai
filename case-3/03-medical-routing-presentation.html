13 · Khuyến nghị

GO chỉ khi red flag recall đạt — bottleneck là expert, không phải API

GO

Gate 1+2 pass · Red flag recall ≥ 98% · Expert ký rubric · 100% medical review trong pilot 5%

NO-GO

Recall < 98% · Diagnostic overreach > 0% · Mock-style under-severity còn xuất hiện trên golden set

Kết luận: Case y tế không thể bỏ expert. API ~$14 không đáng kể; 24h expert (~$1,440) là chi phí cần thiết để pilot an toàn trước khi giảm tỷ lệ duyệt thủ công.
